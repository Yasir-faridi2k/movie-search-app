# 🎬 Movie Search App

A simple movie search app built with React as a learning project.

## What this app does
- Search for any movie by name
- See details like plot, cast, director and IMDB rating
- Save your favorite movies
- Favorites are saved even after refreshing the page

## Technologies I used
- React.js
- React Router DOM
- Context API
- OMDB API

## How to run this project

1. Clone the repo
```bash
   git clone https://github.com/your-username/movie-search-app.git
```

2. Install dependencies
```bash
   npm install
```

3. Add your OMDB API key in:
   - `src/hooks/useMovieSearch.js`
   - `src/pages/MovieDetail.js`

4. Run the app
```bash
   npm start
```

## What I learned building this
- How to fetch data from an API in React
- How to use React Router for navigation
- How to use Context API for global state
- How to store data in localStorage
